# Calcpy 🧮

Una aplicación de escritorio ágil y minimalista desarrollada en **Python** y **Kivy**. Está diseñada específicamente para el personal en Venezuela que necesitan calcular precios rápidamente de manera visual y realizar conversiones automáticas entre dólares (USD) y bolívares (Bs.).

## ✨ Características

* **Cálculo Bifocal Dinámico:** Muestra el total acumulado en dólares de referencia y su equivalente exacto en bolívares al instante.
* **Precisión Financiera:** Algoritmos de redondeo estrictos para asegurar el cuadre exacto de la caja sin imprecisiones por flotantes.
* **Gestión de Lista en Vivo:** Interfaz interactiva para añadir productos mediante buscador y eliminar elementos duplicados o específicos de forma segura en un solo clic.

## 🛠️ Tecnologías Utilizadas

* **Python 3** - Lógica del sistema.
* **Kivy & Kvlang** - Framework para el desarrollo de la interfaz gráfica de usuario (GUI).
* **pyBCV** - API para la extracción automatizada y segura de las tasas cambiarias nacionales.

## 🚀 Instalación y Uso

1. **Clonar el repositorio:**
   ```bash
   git clone https://github.com
   cd Calcpy
   ```

2. **Instalar las dependencias requeridas:**
   ```bash
   pip install kivy pybcv
   ```

3. **Ejecutar la aplicación:**
   ```bash
   python main2.py
   ```

## 📋 Requisitos Previos

* Conexión a Internet (requerido al iniciar la aplicación para la sincronización automática de la tasa cambiaria).
* Python 3.8 o superior instalado en el equipo de escritorio.
